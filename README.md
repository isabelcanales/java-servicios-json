# Consumo de servicios web con JSON en Java

Proyecto realizado en Java que consume un servicio web y procesa los datos recibidos en formato JSON.

## Tecnologías utilizadas
- Java
- JSON
- Servicios web / API

## Descripción
Este proyecto fue desarrollado como práctica del ciclo formativo **Desarrollo de Aplicaciones Multiplataforma (DAM)**.  
El programa realiza una petición a un servicio web, obtiene datos en formato JSON y los procesa para mostrarlos en la aplicación.
